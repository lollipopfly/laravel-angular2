#Laravel & Angular2 boilerplate.#

##Getting Started##
Clone Repo

````
git clone https://github.com/lollipopfly/laravel-angular2.git
````
Create Database

Cd project and run composer install and npm install

````
cd laravel-angular2
composer install
npm install
````

Add .env file to root directory.

Migrate tables

````
php artisan migrate
````

### Gulp ###
````
gulp
````

### Angular 2 ###
you can find angular files in

````
resources/assets/js
````

output files

````
public/js
````

### SCSS ###
Css is written in scss and postcss.

Sass files
````
resources/assets/sass
````

output files

````
public/css
````
